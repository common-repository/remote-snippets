== Description ==

= Display live data coming from a JSON API or webservice on your Wordpress website =

Using this plugin you can fetch data in JSON or CSV format from an external source.

You can then transform it and display it on your Wordpress website using a shortcode.

* Enter the URL of the API or data source and the plugin will retrieve it for you.
* Configurable cache so you do not experience delays or overload the API endpoint.
* Define templates in the Twig template language to transform the raw data to a nice format.
* Use a shortcode to display a template in your pages.

== Installation ==

1. Login to your WordPress installation
2. Install plugin by uploading remote-snippets.zip to `/wp-content/plugins/`.
3. Activate the plugin through the _Plugins_ menu.
4. Go to the RemoteSnippets item in the left bar.
5. Define your templates and configure the URL where to fetch data from.
6. Use the `[remotesnippet]` shortcode to insert the data into your pages.
